## How to Use

1. **Open in Browser** - View SVG files in any web browser
2. **Edit with Design Tools** - Open in Figma, Adobe XD, or Illustrator
3. **Customize Colors** - Edit hex values for brand colors
4. **Export** - Save as PNG, JPG, or PDF
5. **Scale** - SVG formats are infinitely scalable

## Recommended Usage

- **Website Header:** Use at 200px width minimum
- **Social Media:** Scale to 400x400px
- **Print Materials:** 2 inches minimum width
- **Favicon:** Simplify and export at 32x32px
- **App Icon:** Export at 180x180px or 256x256px
